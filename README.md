# Travel Customer App

## 🧩 Description

Travel Customer Web is a responsive web application built with React to centralize all the essential information for participants of an organized trip.

The application provides travelers with easy access to the itinerary, daily activities, services included,  maps, travel tips, useful links, weather information, packing recommendations, all from a single platform.

The project focuses on creating an intuitive user experience, allowing participants to quickly find the information they need before and during the trip. It was developed as a real-world project for travel clients.
